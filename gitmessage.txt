# Conventional Commit Template for DarkLab ✍️
#
# Format: <type>: <short description>
#
# Example:
#   docs: add cron-akmods guide for NVIDIA module monitoring
#   scripts: update check_akmods.sh with better logging
#   fix: correct path in versionlock script
#   chore: remove unused script
#   make: add Makefile target for akmods cleanup
#
# Commit types:
#   docs:     documentation only
#   scripts:  shell scripts, automation
#   make:     changes to Makefile(s)
#   infra:    cron/systemd, timers, config
#   fix:      bugfix or correction
#   chore:    cleanup, refactor, non-feature update
#
# Tip: Use present tense (e.g. “add”, not “added”)
